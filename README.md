# Introduction

This Typst library provides SVG for all the ISO 7010 safety signs, minus country-specific variants and outdated signs.

Thanks to **Wikipedia**, **Wikimedia Commons** and all the **authors** of these SVG for providing the [ISO 7010]("https://fr.wikipedia.org/wiki/ISO_7010") icons under a free licence.

Warning, this package contains 318 SVG files, which is quite a large number : please, import only the safety signs you need for your document.

# Documentation

The documentation of this package is available in manual.typ.

# Tool

This Typst package was generated with the Python script `extract.py`.
